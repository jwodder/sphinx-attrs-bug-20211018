Installation
============
``sphinx-attrs`` requires Python 3.6 or higher.  Just use `pip
<https://pip.pypa.io>`_ for Python 3 (You have pip, right?) to install
``sphinx-attrs`` and its dependencies::

    python3 -m pip install sphinx-attrs
